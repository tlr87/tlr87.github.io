How does JavaScript compare to HTML and CSS?

  • HTML is a static language that is used to as a structure and the content for websites to exist and so data can exist. This is like a stage and actors with in a play.
  • CSS is a way in which the data is presented and positioned on the website. This is like the props and costumes with in a play.
  • JavaScript is the way in which content is changed on the site or presented on based on the directions of the user. This is like a director and special effect within a play.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

  • Control flow is tasks that may happen once, multiple times or not at all.

An example of this is If you were cleaning a house you could need a broom or mop. You may decide to just buy a mop.

  • Loops are tasks that only activate if a situation is present. An example of this is If you were cleaning a house and you have a mop but the floor must be cleaned until the dirt has gone. This could take 3 or more goes of the mop in the same place before it’s cleaned.  

Explain the difference between accessing data from arrays and objects.
Arrays and Objects are different in the way in which they store data.  

  • Arrays can have any data in any order. An example of this could be a junk draw that has not been sorted out. Within that junk draw there may be a box which as more junk in it.  
  • Objects are like phonebooks or a filing cabarets the information is organized in according to number or name.  This in turn could mean that you have file with in a file in the filing cabaret.

Explain what functions are and why they are useful.
In simple terms functions are factories in code that can produce take product or data and convert other data products. This allows us to reproduce the results faster rather than making a new factory all the time.
