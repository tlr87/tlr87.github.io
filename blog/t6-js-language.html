In the JavaScript language, describe all the use cases for each of:

parentheses ()
  • Exacutes a function. E.g. var variable = function()
  • Shows a place to put Inputs to a function. E.g. GenericFunction(parameter 1, parameter 1) 
  • A place to put your arguments when making a for loop. E.g. for (var i = 0; i <10; i++) { do something here.}
brackets []
  • Shows the place to put data in an array. E.g. GenericArray [ 1, 2, “Hello”, true].
  • Indicates an place to where to find an element in an array E.g. var name = ClassNames[0];.
braces {}
  • A place to a put a result in an if statement E.g. if (argument===true) {Result }
  • Indicates a place to where in an object where data is present. E.g. var Dictionay = {Letter:"A", Word:"Ant", Definition: ”Small Insect” };

  • Alterative way in which you can declare a sting except when in JSON where double quotes is monitory.
