What is meant by boxifying design?

This is a way of looking at websites in grid form and deterring the layout of the website and where the elements of the website will be in a grid like structure.

What is the box model?

The box model looks at the properties of individual elements and gives them the ability to be move and appear in the page.

These properties are defined through Margins, Padding and Borders.
