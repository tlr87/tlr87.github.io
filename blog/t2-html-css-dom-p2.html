What is grid based design?

Gird based design is a graphic designers technique to make sure that everything in a document. The idea is to provide elements with guides proportion and breathing room around.

Whats all the hype about responsive webpages?

Responsive webpages look to keep consistent aesthetic and functional experience for the user no matter what the screen size. Web sites should be designed to handle desktop, mobile phone and tablet screens in order to qualify to be a responsive website.

What is semantic structure?

Semantic structure in the web is the way in which we go about and structure our documents with the appropriate tags. Traditionally in HTML this was done with Class (class) or an Identity (id) tag to describe what an element is used for. Now HTML 5 is now using different element types that better define what it can be used for, an example of this can be shown by replacing <div id=”nav”> with <nav>.
